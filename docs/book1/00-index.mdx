# BOOK1：前线

这个位置算作 book1 的封面，本来大可以绘制一个类似杂志封面一样的宣传图，但是我觉得很长一段时间内这都没有必要

下面那个是莫奈的一幅画，并不是出名的那几幅，网络上资料倒是不多，名字叫做睡莲池与玫瑰，也许能搜到吧，我喜欢这幅画上半部分的色彩，这里特指一下是照片中这样的色彩，这种优化过的、鲜艳的色彩

book1 是随机、突发、繁杂的，这和前线阵地很像，故而得名

其本身不稳定，产出的子内容待成熟时将分割开，不再属于 book1

亦或者，全都属于 book1，现在是前线，前线是曾经
