# 基础理论





### 天线效应：

由于金属走线过长吸收带电粒子，导致电压上升，达到一定阈值会击穿场氧化层。(一般16V会被击穿)

**解决：**
1. 靠近栅极的位置采用跳线桥接，在加工时，一个层完成后会将该层电荷引出。
2. 用PN结，反偏击穿，释放电荷

### 软连接：

没有可靠的衬底接触

**解决：**
在器件周围要有沉底(pick up)

### 闩锁效应：

**Q1:** pmos的源极-N阱-Psub,形成PNP
**Q1:** nmos的源极-Psub-N阱,形成NPN
而N阱和衬底均有一定的电阻，因此形成右图所示的正反馈电路，使电路发生短路。

**详细解释：**
当x点注入电流的时候 X点电压上升 Q2导通 R1的通路导通，电流流过R1会产生压降，Y点是Q1的基极电压，Q1的集电极电压>基极>发射极，Q1导通，后使Q2的导通加剧，Q2导通加剧，Q1导通也加剧，形成正反馈，分布电阻很小，这个时候形成大短路，导致器件甚至芯片烧毁。

**解决：**
1. 选择适当的掺杂浓度，降低寄生电阻
2. 采用guard rings 隔离NMOS和PMOS器件。P+ Ring环绕NMOS并接GND, N+ Ring环接PMOS并接VDD。

### 电迁移：

导线上电流较大时，电子的流动会撞击金属线的原子，若发生粒子迁移则会导致断路；若形成粒子堆积则会导致短路发生。

**解决：**
防止发生电迁移，注意查看不同导线的单位电流密度，在设计时要留有裕量。

### 金属应力：

由于金属层制作的时候温度不一致，会产生物理应力，可能导致金属层断裂，

**解决：**
在金属层之间打上狭缝（slot）

### 金属密度：

若金属层覆盖的密度不均匀，在化学or机械抛光时，会出现较软部分形成凹陷，使得一致性变差，严重会导致断裂。

**解决：**
1. 工艺厂会有用自动铺线软件解决密度问题，但是会产生寄生电容，会影响电路特性。
2. 对于关键部分，最好还是手动进行填充，然后做充足的post simulation
