# 键盘

最后的效果是，平时静音，可以选择性地开启蜂鸣器和电磁阀，蜂鸣器的声音类似民国特务电报，电磁阀类似于手机内置的振动器



| 配件         | 备注                         |
| ------------ | ---------------------------- |
| PCB 板卡     | 三模，电磁阀，蜂鸣器         |
| 电磁阀       | 长 28mm，宽 11mm，高 12mm    |
| 蜂鸣器       | 长 9.6mm，宽 9.6mm，高 5.5mm |
| pom 定位板   |                              |
| 钢板卫星轴   | 润油，粉色                   |
| 外壳         | GH60 透明黑                  |
| 电池         | 3000mA                       |
| 2.4G 接收器  |                              |
| poron 夹心棉 | 国产                         |
| 键轴         | 高特 水蜜桃轴v2，静音线性轴  |
| 键帽         | 侧刻灰黑渐变，PBT材质        |

后来，把电磁阀更换成了一堆 iPhone 上面的线性马达，对比起来没那么生硬，而且震源分布到了键盘各处，所以综合体验更好

最大的缺点在于，工作量太大，整个安装过程我花了几个小时

后来还是直接买了一个精致的舒适的便携的键盘，NuPhy Air75 V2
