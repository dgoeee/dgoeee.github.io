# 主板和cpu

## CPU

AMD 的 CPU 实在太适合游戏了，所以我基本没有犹豫

TDP 是散热设计功耗的意思，这个功耗描述很敷衍，不过可以大概了解

| 型号       | TDP（散热设计功耗） | 适配主板           | 价格 | 备注                   |
| ---------- | ------------------- | ------------------ | ---- | ---------------------- |
| R7-7800X3D | 100                 | B650迫击炮、重炮手 | 2300 | 超级大三缓，内存上6000 |
| R5-7500F   | 115                 | B650               | 930  | 大三缓，内存上6000     |
| R5-5600    |                     |                    | 670  |                        |
| I5-13600kf | 185                 | B660               | 1600 |                        |

总之我最后决定上 R5-7500F，其实到这里 CPU 的想法是，我选择差一点的 CPU，然后好一点的主板，等后面多年后 R7-7800X3D 降价，如果仍有需求，再花白菜价提升

或者如果最后预算够，那就直接 R7-7800X3D

## 主板

那么主板就是直接按照 R9-7800X3D 来考虑了，我这里选择 bios 友好的微星 B650M 迫击炮，也就是 MAG B650M MORTAR WIFI 迫击炮

其实 B650M AORUS PRO AX 电竞雕也很合适，综合考虑最后选的电竞雕

| 型号   | 规格  | 核心供电规格 | 价格 | 备注         |
| ------ | ----- | ------------ | ---- | ------------ |
| 迫击炮 | M-ATX | 12相80A      | 1100 |              |
| 电竞雕 | M-ATX | 12相 70A     | 1300 | PCIE插槽优势 |
