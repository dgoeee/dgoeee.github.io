# 尘白禁区笔记

## 肴-冬至配队



### 肴-冬至

![](https://www.cbjq.com/uploadfile/2023/0704/20230704115608906.png)

武器采用专武 `太空骑手`，最好是 2 阶，这个角色太吃专武了，而且专武提升超级大，目测 20% 以上这个级别
| 武器名称 | 等阶 1                                                   | 等阶 2                                                    |
| -------- | -------------------------------------------------------- | --------------------------------------------------------- |
| 太空骑手 | 高热类型伤害提升 18%；装填子弹后，下一次射击伤害提升 60% | 高热类型伤害提升 30%；装填子弹后，下一次射击伤害提升 100% |

普通后勤是 `天岩户小队`，但是实际应用中我选择 `达摩小队`，因为手机端的自瞄和腰射更加契合

| 后勤名称   | 两名成员效果     | 三名成员效果                                                 |
| ---------- | ---------------- | ------------------------------------------------------------ |
| 达摩小队   | 射击伤害提升 24% | 使用常规技后，非瞄准射击伤害提升 46%，持续 5 秒              |
| 天岩户小队 | 射击伤害提升 24% | 暴击后，自身瞄准射击伤害提升 10%，最多叠加 5 层，离场后效果清空 |
