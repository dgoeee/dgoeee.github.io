# 袜子胖次和洗衣机的逻辑和哲学

## 经典话题之袜子和胖次到底能不能用洗衣机来洗

我一直觉得洗衣机拿出来的东西，天然附带“干净”属性

干净这个概念应当算作布尔值，我很难定义什么东西叫有点干净或者说 60% 干净，它就像浅白和深白一样荒谬，这让我想起“美”的反义词并非“丑”而是“美丑不分”

:::note

布尔值是一个逻辑值，只有两种：真（True）和假（False）。在计算机编程中，布尔值经常与动作脚本语句中通过比较控制脚本流的逻辑运算符一起使用。布尔值可以用来进行逻辑判断，例如在for循环中判断是否继续循环。在逻辑中，真值或逻辑值是指示一个陈述在什么程度上是真的。在计算机编程上多称作布尔值。

:::

所以如果觉得洗衣机洗过的东西就是干净的，那袜子和其他衣服应该等价，如果认为袜子不干净，那衣服也是确定的不干净

矛盾点就来了，洗衣机洗完之后衣服它不干净，那还用什么洗衣机

重要前提就是用洗衣机的人应该对清洗效果表示满意所以才会使用洗衣机，用洗衣机本身就是对清洗效果的一种认可，如果不认可清洗效果，然后才是手洗

我在网络上看到有大量比较严谨的测试表明了机洗比手洗更干净，而且领先幅度非常大，此处我认为，如果觉得机洗不干净而采用手洗，这本身就是矛盾的

其次假若一次清洗的效率是 90%，粗略理解洗两次就能 90% * 90% = 99%，总之洗两次就可以干净到很离谱的程度，城市环境下大多数人的衣物也少有传统体力劳动担忧的的衣领袖口难题

在确认机洗袜子是干净的之后，这个问题就迎刃而解了，其次若是讨论袜子和衣服混洗的传染问题，也同样适用

就算是锹一铲子沙土和衣服混洗，只要机洗结束拿出来的衣服是干净的，其实无所谓之前有多脏

对于病菌等，不要以为只有直接的接触才是危险的，如果自身免疫低下，它们想要入侵完全不用等待你将之和衣服放到一起，然后经历洗衣机的各种搅拌和几道换水，太麻烦了

它们大可以直接从你正在穿的衣服上着手入侵，亦或是空气传播效率也非常夸张，如果详细看过强光下的灰尘，你会发现袜子的灰尘传染整个房间只需要轻轻弹一下

> 对于家长们来说他们并不需要知道电子游戏是什么，它会不会造成近视，它会不会上瘾，他们只需要一个背锅侠，一个可以掩盖家庭教育失败、学校教育失败、社会教育失败的东西，现在它叫游戏，十五年前它是早恋、三十年前它是偶像、三十五年前它是香港电影、四十年前它是武侠小说。我们最终活成了我们当初所讨厌的样子，用其他掩饰自己的失败，屠龙的少年最终成为了恶龙。在这失败的教育下，他们干翻了电子游戏，那么下一个“替罪羊”是什么？

所以我后来的劝告是，要么你承认衣柜里所有的衣服都是脏的，要么就不要觉得机洗出来的东西不干净，如果洗衣机确实不给力，可以洗两次

而且请相信现代的洗衣机处理城市衣物的能力，同时那群瞎讲究的人压根就没有真正很脏的衣服
