## [blog/legacy](/blog/legacy)

记录各种思绪



顺便提一句：

- heritage侧重的是“传承的遗产”（品质、传统、特点），比如文化遗产这种几代人或者说甚至千百年的东西
- inheritance指“得到的遗产”，侧重于从继承者角度继承的东西，“我的遗产”指父辈的东西
- legacy则特指“留下的遗产”，侧重于从给予者角度遗留的东西，“我的遗产”指我自己的东西
